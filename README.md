# Recursos-humanos-program.
## Programa de gestión en RRHH:
  Sistemas de Recursos Humanos para una empresa de tecnología, que permite centralizar toda la información de los colaboradores, la cual integra información personal y académica del empleado, evaluando el bienestar emocional y profesional del entorno en el cual trabaja.
## Problema que resuelve:
  -	Desconexión entre Perfil y Desempeño.
  - Subjetividad en la Evaulución.
  - Falta de manitoreo de promociones.
## Usurios obejetivos:
  - Analistas de Recursos Humanos (HR Partners).
  - Gerentes y Líderes de Equipo.
  - Administradores de Talento (Talent Management).
